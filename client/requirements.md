## Packages
framer-motion | Smooth page transitions and UI animations
date-fns | Formatting dates in Arabic locale
clsx | Conditional class merging
tailwind-merge | Conflict-free tailwind class merging
react-hook-form | Form state management
@hookform/resolvers | Zod integration for forms
zod | Schema validation

## Notes
- The application uses a "literary" aesthetic with warm paper-like backgrounds.
- RTL (Right-to-Left) direction is mandatory for Arabic text.
- Fonts: 'Amiri' for headings/body, 'Noto Sans Arabic' for UI elements.
- PDF export is handled via native browser print styling.
